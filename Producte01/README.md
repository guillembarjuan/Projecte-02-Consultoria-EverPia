# P01 – Kanban & Compte de GitHub

## 📌 Descripció del producte

Aquest producte forma part del **Projecte 02 – Consultoria EverPia** i combina dues eines fonamentals per treballar com a consultor IT professional:

- **Tauler Kanban amb Microsoft Planner** – per a la gestió visual de tasques de l’equip.
- **Compte i repositori personal de GitHub** – per a l’espai individual de treball, documentació tècnica i control de versions.

L’objectiu és garantir que cada grup i cada alumne disposi d’un sistema d’organització clar, transparent i traçable, replicant la metodologia de treball d’una consultora tecnològica real.

---

## 🎯 Objectius específics

- Disposar d’una eina de coordinació col·laborativa (Planner) amb el backlog complet del projecte.
- Tenir un espai individual a GitHub per allotjar la documentació, els informes i els productes tècnics.
- Aplicar la metodologia Kanban per seguir l’evolució de les tasques (Backlog → En curs → En revisió → Fet).
- Reforçar el compromís amb la transparència, l’organització i la traçabilitat del treball.

---

## 🧩 Part 1 – Tauler Kanban (Microsoft Planner)

Vam crear un tauler a **Microsoft Planner** amb les columnes estàndard de Kanban:

- **Backlog** – tasques pendents del projecte (T01, T02, …, T10).
- **En curs** – tasques en desenvolupament.
- **En revisió** – tasques pendents de validació per part de l’equip o del client.
- **Fet** – tasques completades i acceptades.

## 💡 Aprenentatges

- Ús pràctic de **Kanban** com a metodologia àgil per organitzar tasques en equip.
- Utilització de **Microsoft Planner** per gestionar un backlog, assignar tasques i visualitzar el flux de treball.
